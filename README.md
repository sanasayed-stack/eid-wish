# 🌙 Eid Mubarak Greeting Webpage

A soft, aesthetic Eid greeting webpage designed with a calm pastel theme and elegant layout.

## ✨ Features
- Minimal and clean design
- Pastel color theme (pink/soft tones)
- Centered Eid message
- Decorative lantern elements
- Simple and lightweight (pure HTML)

## 🛠️ Tech Used
- HTML (single file project)
- Inline styling (no external CSS or JS)

## 💡 Purpose
This project was created as a festive Eid greeting card that can be shared or hosted online.

## 🚀 How to Use
1. Download or clone the repository
2. Open the `.html` file in your browser
3. Customize the message or design if needed
